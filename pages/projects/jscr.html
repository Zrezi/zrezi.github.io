This was a short project to basically store a local copy of a Javascript comment parser tool. I decided to throw some CSS on top of what is essentially a direct copy (albeit earlier version) of this. It's pretty simple: put text in, go line by and line and use regex parsing to detect which sections of the text represent comments (i.e. between /* */ and //, but a bit more involved than that), and erase them from the line of text. Append that parsed line of text to a big var that holds the rest of the text's parsed code. Then output that in the other box.
